1 Откройте терминал и перейдите в корневую директорию проекта «L12»
   командой «cd L12»
2 Активируйте виртуальное окружение «venv»
   командой «\venv\Scripts\activate»
3 перейдите дериктории к проекта «lyceum »
   командой «cd lyceum»
4 Запустите проект в «dev» режиме
   командой «python manage.py runserver»
